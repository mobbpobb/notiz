# Windows Server 2022 – Storage Spaces

Diese kleine Dokumentation sammelt einfache Übungen zu **Storage Spaces** in Windows Server 2022.
Der Fokus liegt auf drei Varianten:

- Simple Layout (ähnlich RAID 0)
- Mirror (ähnlich RAID 1)
- Parity (ähnlich RAID 5)

## Hinweis

Die Begriffe RAID 0, RAID 1 und RAID 5 werden hier nur zur **einfachen Einordnung** benutzt.
Technisch geht es in diesen Übungen um **Windows Storage Spaces** und **Speicherpools**.
