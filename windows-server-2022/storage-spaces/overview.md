# Datenträger / Storage Spaces

Diese Seite fasst drei einfache Labore zu **Storage Spaces** zusammen.
Die Struktur orientiert sich am Thema **Datenträger verwalten** aus dem Lehrbuch.

## Überblick

### Simple Layout

Simple verteilt Daten über mehrere Datenträger.
Es ist schnell, aber es gibt **keine Redundanz**.

### Mirror

Mirror speichert Daten gespiegelt.
Dadurch bleiben Daten auch bei einem Ausfall weiter erreichbar.

### Parity

Parity speichert Daten mit Paritätsinformationen.
Dadurch ist ein Kompromiss aus Kapazität und Redundanz möglich.
